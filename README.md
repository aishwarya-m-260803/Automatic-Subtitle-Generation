## Project Overview

Automatic Subtitle Generation is a web-based application that takes audio input and generates subtitle files (`.srt`) automatically. The project demonstrates how speech-to-text concepts can be integrated with a web interface to build a practical and real-world application.

The application includes a user-friendly interface where users can upload audio files and view the generated subtitles.

---

## Key Features

- Upload audio files through a web interface
- Automatically generate subtitle (`.srt`) files
- Display processed results through HTML templates
- Organized frontend using HTML, CSS, and images
- Backend logic handled using Python
